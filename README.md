## How to use TP files

![Preview](howtouse.png)

1. Save the TP text file to:

```text
C:\Program Files\Neverness To Everness\Client\WindowsNoEditor\HT\Binaries\Win64
```

2. Open pipsi
3. Go to the `Teleports` tab
4. Enter the TP file name in the `File Name` field
5. Load the file
